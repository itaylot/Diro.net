# סוכן חיפוש דירות — שכונה ב', באר שבע

מחפש אוטומטית דירות של 2-3 חדרים להשכרה בשכונה ב' בבאר שבע, עד 3,500 ₪/חודש.
שולח אימייל לכתובת itayl1998@gmail.com כשמוצאות דירות חדשות.

## הגדרה ראשונית (פעם אחת)

### 1. התקן Python
הורד מ-https://python.org (גרסה 3.11+)

### 2. התקן ספריות
```
pip install -r requirements.txt
```

### 3. הגדר Gmail לשליחת מיילים
1. כנס ל-myaccount.google.com
2. אבטחה ← אימות דו-שלבי (הפעל אם לא פעיל)
3. חפש "App Passwords" ← צור סיסמה לאפליקציה
4. פתח את `apartment_agent.py` ועדכן:
   ```python
   EMAIL_SENDER   = "your_gmail@gmail.com"
   EMAIL_PASSWORD = "xxxx xxxx xxxx xxxx"   # 16 תווים
   ```

> אם לא תגדיר מייל — הסוכן פשוט ידפיס את הדירות לטרמינל

### 4. הרץ את הסוכן
```
python apartment_agent.py
```

הסוכן בודק כל 30 דקות ושולח מייל רק על דירות חדשות שלא ראה קודם.

## שינוי הגדרות

בתוך `apartment_agent.py`:
| משתנה | ברירת מחדל | הסבר |
|-------|-----------|------|
| `MAX_PRICE` | 3500 | מחיר מקסימלי |
| `MIN_ROOMS` / `MAX_ROOMS` | 2 / 3 | טווח חדרים |
| `interval_minutes` | 30 | כמה דקות בין בדיקות |
